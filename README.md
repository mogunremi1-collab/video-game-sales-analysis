# 🎮 Video Game Sales Analysis

Exploratory Data Analysis on **Video Game Sales** using Python, Pandas, Matplotlib & Seaborn.

![Video Game Sales Dashboard](dashboard.png)

## 📊 Project Overview

This project performs a comprehensive analysis of video game sales data from 1980 to 2024. It explores trends, regional performance, genre popularity, and platform success to derive meaningful business insights.

### Key Questions Answered:
- Which genres and platforms generate the most revenue?
- How have video game sales evolved over the years?
- Which regions contribute the most to global sales?
- What is the relationship between critic/user scores and sales?
- Top performing games and publishers

## 🛠️ Technologies Used

- **Python 3**
- **Pandas** – Data cleaning & manipulation
- **Matplotlib & Seaborn** – Data visualization
- **Jupyter Notebook** – Interactive analysis
- **NumPy**
