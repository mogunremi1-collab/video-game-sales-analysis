# 🎮 Video Game Sales Analysis

**Exploratory Data Analysis on Video Game Sales (1980–2024)** using Python, Pandas, Matplotlib & Seaborn.

![Dashboard Preview](dashboard.png)

## 📊 Project Overview

This project performs a comprehensive analysis of **video game sales data** spanning from 1980 to 2024. It uncovers key trends, regional performance, genre popularity, platform success, and the relationship between critic scores and sales.

### Key Questions Answered:
- Which genres and platforms generate the most revenue?
- How have video game sales evolved over the years?
- Which regions contribute the most to global sales?
- What is the relationship between critic/user scores and sales?
- Who are the top performing games and publishers?

## 🛠️ Tech Stack

- **Python 3**
- **Pandas** – Data cleaning & analysis
- **Matplotlib & Seaborn** – Data visualization
- **Plotly** – Interactive charts
- **Jupyter Notebook** – Interactive analysis
