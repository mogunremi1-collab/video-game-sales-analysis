📊 Project Overview

This project analyzes 64,000+ video game titles to uncover trends in global sales, platform dominance, genre performance, and the correlation between critic scores and commercial success. Built with Python, Pandas, Plotly, and Streamlit.

📂 View GitHub Repo 🚀 Launch Live Dashboard

💡 Key Insights

🎯 Genre Dominance: Action & Shooter games drive ~45% of total sales.
🌍 Regional Split: North America accounts for 52% of global sales.
📈 Platform Peaks: PS2 & PS4 eras dominated; Switch shows strongest post-2017 growth.
⭐ Quality vs Sales: Only 28% of top-selling games have critic scores >9.0.

🛠️ Tech Stack

Python 3 • Pandas • Plotly • Streamlit • Jupyter

🔗 Dataset: Kaggle Video Game Sales (1980-2024) (CC0 Public Domain)
